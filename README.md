# AskTechBro 👨🏿‍💻

An AI-powered chat app featuring TechBro — your senior African tech mentor.
Ask anything about code, tech careers, or software concepts and get real,
no-nonsense answers with African context and flavor.

## Live Demo

[asktechbro.vercel.app](https://asktechbro.vercel.app)

## Features

- Real-time streaming AI responses
- Persistent conversation history
- Mobile responsive design
- TechBro persona — direct, smart, encouraging

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Groq API (llama-3.3-70b-versatile)

## Running Locally

1. Clone the repo
   git clone https://github.com/Davcid1572/asktechbro.git

2. Install dependencies
   npm install

3. Create .env.local and add your Groq API key
   GROQ_API_KEY=your_key_here

4. Start the dev server
   npm run dev

5. Open http://localhost:3000
